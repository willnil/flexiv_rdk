Compute and get the Coriolis/centripetal matrix for the generalized coordinates, i.e. joint space.

Returns
\( \mathbb{R}^{Dof \times Dof} \) Coriolis/centripetal matrix \( C(q,\dot{q}) \)
Note
Use updateModel() to update robot states first before calling this function
Coriolis matrix factorization
The factorization of the Coriolis matrix C is not unique, and this API is using the factorization method found in "A new Coriolis matrix factorization", 2012 by M. Bjerkend and K. Pettersen
