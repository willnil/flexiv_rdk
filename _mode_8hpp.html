enum flexiv::Mode

Operation modes of the robot.

Enumerator
MODE_IDLE 

The robot holds position and waits for commands.

Note
This is the default mode upon initialization
MODE_JOINT_POSITION 

Joint position control.

Note
Real-time control mode running at 1kHz
MODE_JOINT_TORQUE 

Joint torque control.

Note
Real-time control mode running at 1kHz
MODE_PLAN_EXECUTION 

Execute the plan specified by user.

Note
Non-realtime control mode. The user selects a plan and executes it. Upon completion, the user can send another plan
MODE_PRIMITIVE_EXECUTION 

Execute the primitive specified by user.

Note
Non-realtime control mode. The user sends a primitive command with the name and the associated parameters and executes it. Upon completion, the user can send another primitive command
MODE_CARTESIAN_IMPEDANCE 

Cartesian impedance control.

Note
Real-time control mode running at 1kHz
